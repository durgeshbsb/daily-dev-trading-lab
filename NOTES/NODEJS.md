1. Array index out of bounds вҖ” trying to access the seventh element of the array when only six are available
2. Syntax errors вҖ” failing to close the curly braces while defining a JavaScript function
3. Reference errors вҖ” accessing a function or variables that are not defined
4. Deprecation errors and warnings вҖ” calling an asynchronous function without a callback
5. Type error вҖ” x object is not iterable
6. Failing to handle operational errors

> Operational errors are issues during runtime due to external factors that can interrupt the programвҖҷs normal flow
1. Unable to connect server/database
2. Request timeout
3. Invalid input from the user
4. Socket hang-up
5. 500 response from a server
6. File not found

> system errors usually occur when an application violates an operating system constraint. For example, a system error will occur if an application attempts to read a file that does not exist.
